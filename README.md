# CMSC 207 Mini-Project: Personal Web Profile

## Overview
This is a beginner-level multi-page personal web profile built with HTML, CSS, and JavaScript.
It uses a Twitter-inspired interface with a fixed top header, circular profile image, and tabs
that switch between pages. The project uses hardcoded information and has no backend.

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (Twitter-like layout, tabs, hover effects, responsive media query)
- JavaScript (button click interaction with safe DOM guards)

## Project Structure
- `index.html` - redirect entry page to `about.html`
- `about.html` - About tab/page
- `hobbies.html` - Hobbies tab/page + JavaScript fun fact interaction
- `contact.html` - Contact tab/page
- `style.css` - shared styling and responsive rules for all pages
- `script.js` - shared JavaScript file

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any web browser.
3. Use tabs to navigate between About, Hobbies, and Contact pages.
4. On the Hobbies page, click the fun fact button.

## Required Sections Included
- Fixed header shell with profile info and tab navigation
- About page with introduction and profile details
- Hobbies page with at least three hobbies and descriptions
- Contact page with email and social links
- JavaScript interaction on the Hobbies page

## Short Reflection
Working on this mini-project helped me practice the basics of semantic HTML, CSS styling,
and simple JavaScript interaction. The main challenge was making sure the layout stayed
clean and readable on both desktop and mobile screen sizes.

I learned how to connect a button click to dynamic text updates and how anchor links improve
single-page navigation. I also became more confident organizing files and keeping code simple
and beginner-friendly.
